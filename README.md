# HavaDurumu - Hava Durumu PWA

  Tam tesekkullu Progressive Web App (PWA) hava durumu uygulamasi.

  ## Ozellikler

  - Anlik hava durumu (sicaklik, nem, ruzgar, basinc, bulutluluk)
  - 7 gunluk hava tahmini
  - 24 saatlik saatlik tahmin
  - Hava kalitesi indeksi (AQI)
  - UV indeksi
  - Gun dogumu / gun batimi
  - Gorus mesafesi
  - Sehir arama (otomatik tamamlama)
  - GPS ile konum tespiti
  - Favori sehirler
  - Arama gecmisi
  - Karanlik / aydinlik tema
  - Celsius / Fahrenheit donusumu
  - Hava durumu animasyonlari (gunes, yagmur, kar, firtina, bulut)
  - Offline destek (Service Worker)
  - Telefona kurulabilir (PWA)
  - Tamamen responsive tasarim

  ## Teknolojiler

  - Saf HTML, CSS, JavaScript (framework yok)
  - Open-Meteo API (ucretsiz, API anahtari gerektirmez)
  - Service Worker (offline destek)
  - Web App Manifest (PWA)
  - CSS Animations & Glassmorphism
